==================================================================================
=                                Workshop FSharp                                 =
==================================================================================

Voor deze workshop is nodig:
* Circa 2 Gb schijfruimte voor de Docker containers
* De laatste versie van VS Code
* De VS Code extension "Remote - Containers", deze geeft een groene button linksonder
* De laatste versie van Docker Desktop voor Windows
* Docker moet werken met Linux-gebaseerde containers (zie Docker settings)

De omgeving kan worden opgestart door de folder Remote te openen in VS Code:
1.  Druk in VS Code op de groene knop linksonder;
2.  Kies "Open Folder in Container"
3.  Selecteer de folder waar deze README.txt in staat.

Of open de folder in VS Code, bij de vraag kies "Reopen in Container".


Voorbereiding
---------------------------------------------------------------------
Start de omgeving een keer op voordat de Workshop begint, de eerste
keer neemt het ongeveer een kwartier in beslag.


Workshop Inhoud
---------------------------------------------------------------------
Folder WorkshopFSharp bevat de hoofdlijn van deze workshop:
- 01 - basis.fsx
- 02 - Structuren.fsx
- 03 - Net Interactive Notebook.ipynb
- 04 - Werken met een DSL.fsx

Folder "Uitstapjes" bevat verdieping en vanuit de hoofdlijn wordt hiernaar verwezen.

Folder "Antwoorden" bevat de antwoorden op diverse oefeningen.
